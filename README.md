# SubTracker

A mobile application for managing recurring subscriptions and free trials, built with Expo and React Native.

## Features

- **Subscription Management**: Track weekly, monthly, and yearly subscriptions.
- **Free Trial Tracking**: Mark subscriptions as free trials and get notified before they expire.
- **Notifications**: Automatic reminders 24 hours and 2 hours before a free trial ends.
- **Expense Overview**: See your total monthly expense at a glance.
- **Dark Mode**: Sleek "Midnight Minimalist" UI.
